#WorldCup 2026 Predictor

Pick winners for the whole World Cup. Group stage, knockout bracket, champion.

## How to use
- Tap any match to pick winner
- Tap "Draw" for a tie
- Standings update automatically
- Bracket fills itself as you go

## Features
- All 103 matches (72 group + 31 knockout)
- Saves your picks automatically
- Share your bracket with a link
- Music player with FIFA songs
- Works on phone & desktop

## Files
- `index.html` - the app
- `script.js` - all the logic
- `styles.css` - styling

## Run it
Just open `index.html` in a browser. No installation needed.
